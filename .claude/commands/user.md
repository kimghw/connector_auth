# 사용자 시나리오 기록 지침

현재 윈도우에서 사용자가 질의한 내용을 정리해서 다음 아래의 파일에 저장한다.

**저장 위치**: `/home/kimghw/Connector_auth/.claude/preprompts/usersenario.md`

## 기록 기준

### 기록해야 하는 내용
- 구체적인 기능 요청 (예: "대시보드 추가", "병합 기능 구현")
- 버그 수정 요청
- 아키텍처 변경 요청
- 새로운 API/엔드포인트 추가 요청
- 설정/옵션 변경 요청

### 기록하지 않는 내용
- "진행해 주세요", "실행해 주세요" 같은 일반적인 요청
- 단순 확인 질문
- 기존 기능에 대한 질문 (How-to)

## 기록 형식

```markdown
### YYYY-MM-DD: [작업 제목]

#### 요청 사항
- [구체적인 요청 내용]

#### 구현 완료 항목
- [완료된 항목 목록]

#### 수정된 파일
| 파일 | 수정 내용 |
|------|----------|
| `파일경로` | 변경 설명 |
```

## 주의사항
- 질의에 대한 답변을 작성할 필요 없이, **사용자의 요청사항만** 작성
- 깔끔하게 작성
- 최신 세션이 파일 상단에 오도록 작성
